Installing SAMS on your machine

  • Requirement:  A Java Runtime Environment (JRE), version 1.4 or newer, is required on your system. You can download a JRE from http://java.sun.com/j2se/1.4/download.html.Scroll down to "Download J2SE v 1.4.2_04" and choose the "download" link under JRE column. Then follow the instructions according to your operating system (Windows, Linux, Solaris).
  • Download the SAMS installer: install-sams-3.1.jar .
  • Run the installer: Note that the SAMS installer is an "executable JAR file."  On some systems it will suffice to double-click the installer from a file explorer window to execute it. If this does not work, then you can run the installer manually as follows:
    • Open a command window (DOS prompt / console / terminal)
    • Enter the following commands:
      • cd <directory-where-you-downloaded-the-installer>
      • java -jar install-sams-3.1.jar
  • Follow the SAMS installer instructions: Basically you need to specify a directory where the SAMS system will be installed.
