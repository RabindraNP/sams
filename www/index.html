Installation

  • Requirement: Since SAMS is a Java application, a Java Runtime Environment (version 1.4 or newer) is required on your system.
  • Download the SAMS installer: install-sams-3.0.jar (updated 08/25/03)
  • Run the installer by double-clicking on the installer file.
  • Follow the instructions.
