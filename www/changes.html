Changes in 3.0

  • Internally, SAMS now uses a different, more flexible scheme to store databases. Among other things, this now allows the user to specify the directory where a SAMS database is located.
  • More than one database can be open simultaneously.
  • Elements in the spectra table can now be sorted and filtered.
  • Grouping creation has been reimplemented to allow for more general criteria. A grouping is specified as a list of expressions Expressions may have different forms, the name of an attribute being one very common.
  • Better progress report in long operations: shows what is being done, involved signatures, and error messages.
  • "paste" is now a real copy of signatures. ("paste reference" is no longer available.)
